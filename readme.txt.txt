
This repository is the official implementation of [Bayesian Uncertainty Matching for Unsupervised Domain Adaptation]

##To install requirements:
pip install -r requirements.txt


## Training:
1. Download datasets and put in: ./data/
2. Training and test:  ./scripts/mnist2000-usps.sh



## Pre-trained Models
You can download pretrained models and dataset here:
1. AlexNet weights: http://www.cs.toronto.edu/~guerzhoy/tf_alexnet/
2. Datasets: https://github.com/jindongwang/transferlearning/tree/master/data
